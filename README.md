Fungsi:
Menjumlahkan dua buah matriks dengan ukuran sama.

Cara kerja:

Pengguna menentukan ukuran matriks (baris dan kolom).

Menginput elemen matriks A dan B.

Program menambahkan setiap elemen yang posisinya sama (C[i][j] = A[i][j] + B[i][j]).

Matriks hasil (C) ditampilkan di layar.
